Description

A workshop that aims to introduce children to computer programming ("coding") and artificial intelligence.

Activities

  • Shoot the bug - A game where the player has to shoot a bug that appear on the screen. The game is implemented in Scratch, a visual programming language developed by the MIT Media Lab.
  • Face finder - A program that detects faces in a picture. The program is implemented in Scratch, a visual programming language developed by the MIT Media Lab.
